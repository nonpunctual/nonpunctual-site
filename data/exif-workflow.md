# EXIF Metadata Workflow for Scanned Photos

For your own digital photos, EXIF is trustworthy,

## The Template

`exif-template.json` in this directory contains the EXIF fields most useful for
personal photo archiving.

The full EXIF spec has hundreds of additional fields (manufacturer-specific tags, 
obscure technical imaging data, etc.) that are either auto-generated by cameras or
have no practical use. 

It is safe to delete JSON fields you have no data for. Do not leave fields
blank if you have no data just remove them.

## Workflow for single photo

0. To wipe all existing EXIF before writing your own data:

   `exiftool -all= mom.jpg`

1. Copy the template:

   `cp data/exif-template.json mom.json`

2. Fill in what you know. Example for a 1972 kitchen photo:

```json
   - DateTimeOriginal: "1972:01:01 00:00:00" (year only — use Jan 1 as placeholder)
   - Artist: "Dad"
   - City: "Golden Valley"
   - State: "Minnesota"
   - Country: "United States"
   - ImageDescription: "Mom in the kitchen."
   - PersonInImage: ["Mom"]
```

3. Write the metadata to the image:

   `exiftool -json=mom.json mom.jpg`

## Date Format

ExifTool uses YYYY:MM:DD HH:MM:SS — note the colons in the date portion.

If you only know the year: 1972:01:01 00:00:00

If you know the year and month: 1972:06:01 00:00:00
